# FC_Marlon_Database_Management_System
DBMS project for a football club(FC_Marlon)

FC Marlon Football Club Database Management System (DBMS)

Project Overview
This project is a custom-designed database management system tailored for FC Marlon football club to centrally manage player, staff, contract, and medical information. It streamlines operations such as squad management, contract monitoring, loan tracking, and medical status reporting to support informed decision-making and operational efficiency.

Key Features
Centralized storage of all club data including players, staff, contracts, and medical records.

Automated alerts for contract expirations and loan durations to reduce administrative errors.

Real-time analytical reporting on squad composition, market values, and player statuses.

Role-specific data access via simplified queries and database views for management, coaches, and staff.

Data integrity through normalization and secure handling of sensitive information.

Technologies Used
Relational database (SQL-based) with normalized schema (3NF).

Advanced SQL queries and views to support core club operations.

Automated notification logic embedded in database procedures.

Installation
Set up the database schema as per the provided SQL scripts.

Load initial data for players, staff, positions, nationalities, and loan histories.

Configure triggers or scheduler for automated alerts if applicable.

Usage
Access predefined views to retrieve player lists by position, contract alerts, loaned player details, and medical fitness summaries.

Generate analytical reports for squad value and contract management.

Authorized personnel can perform CRUD operations adhering to club roles.
